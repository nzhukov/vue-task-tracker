
### Почему при переходе со странице «Об авторе» не отображались другие страницы?

Ошибка при переходе со странице "Об авторе" на другие страницы была связана с тем, что ошибочно отрабатывала анимация transition из-за разницы в структурах View-страниц: 

```html 

<section class="hero">
    <div class="hero-content">
    <h1 class="roboto-condensed-section-title"> Об авторе</h1>
    <p class="hero-subtitle">
        Nikolai Zhukov
    </p>
    </div>
    <p class="error-path">
    Запрошенный путь: <code>{{ currentPath }}</code>
    </p>
</section>
```
После обертывания данного фрагмента в дополнительный div, структура страницы ```AboutView.vue``` стала идентична другим страницам и все стало отображаться корректно (см. код).
